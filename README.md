# 📊 Análisis de Ventas - Alura Store

Este proyecto forma parte del **Challenge de Data Science de Alura LATAM**.

El objetivo es analizar el desempeño de cuatro tiendas de la cadena **Alura Store** para identificar cuál presenta el menor rendimiento y debería venderse para financiar un nuevo emprendimiento.

---

# 🛠 Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

---

# 📂 Dataset

Los datos utilizados corresponden a registros de ventas de cuatro tiendas e incluyen información como:

- Producto
- Categoría del producto
- Precio
- Costo de envío
- Calificación de clientes

---

# 📊 Análisis realizado

Durante el proyecto se analizaron las siguientes métricas:

- Ingresos totales por tienda
- Categorías de productos más vendidas
- Promedio de calificaciones de clientes
- Productos más vendidos
- Costo promedio de envío

Se realizaron visualizaciones utilizando **Matplotlib** para facilitar la interpretación de los resultados.

---

# 📈 Visualizaciones

Se crearon diferentes gráficos para representar los datos:

- Gráfico de barras de ingresos por tienda
- Gráfico de barras de calificaciones promedio
- Gráfico de categorías de productos
- Gráfico de productos más vendidos
- Gráfico de costo promedio de envío

---

# 📌 Conclusión

Después del análisis de los datos se determinó que **la Tienda 4 presenta el menor rendimiento en términos de ingresos**, sin mostrar ventajas significativas en otras métricas analizadas.

Por esta razón, se recomienda que el Sr. João considere **vender la Tienda 4**, permitiendo así liberar capital para invertir en un nuevo emprendimiento.
